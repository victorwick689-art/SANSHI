# SanShine Organisation Website

This is the official multi-page website for **SanShine Organisation**, with full M-Pesa API integration.

## 🚀 Deployment

### Frontend (Client)
- Framework: React + Vite
- Hosting: Vercel
- Env File: `client/.env`

### Backend (Server)
- Framework: Node.js + Express
- Hosting: Render
- Env File: `server/.env`

## ⚙️ Setup

1. Clone this repo into your GitHub account.
2. Rename `.env.example` → `.env` in both `client/` and `server/`.
3. Add your Safaricom **Daraja API Keys** in `server/.env`.
4. Deploy:
   - Client → [Vercel](https://vercel.com)
   - Server → [Render](https://render.com)
